# 배터리 상태 조회

## 브라우저 지원 범위
- Chrome / Edge / Opera / Android 에서 지원한다. 
- [Browser compatibility](https://developer.mozilla.org/en-US/docs/Web/API/Battery_Status_API#browser_compatibility)
