# 더넥스트웹리서치랩 소개
더넥스트웹리서치랩은 세로토닌이 생기는 담백한 프런트엔드 기술 블로그를 운영하기 위해 노력합니다.
콘텐츠에는 근 미래에 웹 표준이 될 기술과 10년 간 20개 이상의 서비스를 경험한 프런트엔드 노하우를 담았습니다.
